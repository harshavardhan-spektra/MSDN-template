# Exercise 3 – FastAPI Server Mode
**Estimated Duration:** 70 Minutes

## Objective
Host Agent Memory as a FastAPI service and access it remotely.

### Task 1 – Configure the FastAPI Service
1. Navigate to the FastAPI server project.
2. Configure Agent Memory settings.
3. Verify the configuration.

### Task 2 – Start the Memory Server
1. Launch the FastAPI application.
2. Verify the REST endpoints are available.
3. Review startup logs.

### Task 3 – Connect Using MemoryServiceClient
1. Configure the client endpoint.
2. Establish a connection.
3. Verify successful communication.

### Task 4 – Access Memory Remotely
1. Send memory requests through the client.
2. Store new memories.
3. Retrieve existing memories.
4. Validate multi-turn conversations.

### Task 5 – Validate Remote Integration
1. Confirm end-to-end communication.
2. Verify memory persistence.
3. Review the remote integration architecture.
