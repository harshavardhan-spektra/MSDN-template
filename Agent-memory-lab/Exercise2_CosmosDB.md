# Exercise 2 – Azure Cosmos DB Production Demo
**Estimated Duration:** 90 Minutes

## Objective
Transition the same application from SQLite to Azure Cosmos DB.

### Task 1 – Configure Azure Cosmos DB
1. Review the provided Azure Cosmos DB resources.
2. Update endpoint, database, and authentication settings.
3. Save the configuration.

### Task 2 – Switch Backend
1. Change only the backend configuration.
2. Leave application code unchanged.
3. Restart the application.

### Task 3 – Run the Cosmos DB Demo
1. Execute the production demo.
2. Verify successful connectivity.
3. Review application logs.

### Task 4 – Interact with the AI Agent
1. Continue conversations.
2. Add new memories.
3. Ask context-aware questions.
4. Verify responses are based on stored memory.

### Task 5 – Validate Cosmos DB Memory
1. Open the Azure Cosmos DB containers.
2. Verify memory documents.
3. Compare behavior with SQLite.
